## 安装

在 Astrbot 应用市场点击右下角 + 号，选择`从链接安装`，复制粘贴本仓库 URL 并点击安装即可。

## 使用

- 自动开盒新群友（需在配置里添加群聊白名单）
- 指令调用：

```plaintext
/box [@目标]
/box [QQ号]
```
